## What

A preprint is a complete manuscript posted to a public server (bioRxiv, medRxiv, arXiv, Research Square, SSRN) before — or during — peer review. Preprints carry a DOI, are time-stamped, citable, and indexed by Europe PMC and Google Scholar.

## When

Most authors post a preprint at the same time as journal submission. For time-critical work (pandemics, public-health alerts) post first and submit shortly after.

## How

1. Confirm the chosen target journal accepts prior preprints (most do — check SHERPA RoMEO or the journal's policy page). Some surgical and clinical-trial journals still don't.
2. Pick the right server: bioRxiv (biology), medRxiv (clinical), arXiv (physics/CS/quant-bio), Research Square (multidisciplinary), SSRN (social sciences). medRxiv requires institutional verification for clinical work.
3. Choose a licence at upload (CC BY for maximal reuse; CC BY-NC if your funder allows).
4. Upload PDF + supplementary; the server screens (24-72 hours) for safety/quality, not novelty.
5. Cite the preprint DOI in your cover letter, ICMJE form, and later journal submissions.

## Journal compatibility

- Almost all biomedical journals (BMJ, Lancet, NEJM, JAMA family, Nature, eLife) accept preprints. NEJM and Lancet require disclosure in the cover letter.
- Some surgical and society journals are restrictive — check before posting.
- "Scooping" protection: most journals consider the preprint as the first public disclosure for priority claims.

## Updating

You can post revised versions to the preprint server (v1, v2). When accepted at a journal, add a comment with the DOI of the published version so readers can find the version of record.

## Citation

Cite preprints with the explicit "[preprint]" tag and the server DOI:

> Smith J, Lee R. Outcomes of laparoscopic vs open repair (preprint). medRxiv 2024. doi:10.1101/2024.04.12.589371.

Once the peer-reviewed version is published, prefer the published citation but readers should still be able to find the preprint version.

## Pitfalls

- Posting raw, unproofread drafts. Preprints are read; treat them as you would a journal submission.
- Failing to disclose the preprint to the journal — many revoke acceptance for non-disclosure.
- Posting on a poorly-indexed server that journals do not recognise as a preprint server.
- For clinical-trial preprints, omitting trial registration ID and the abstract structured fields medRxiv requires.
